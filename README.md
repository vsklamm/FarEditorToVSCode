# Far Editor -> VSCode

## Installation

### Auto

Run `install.bat`.

### Manual 

Copy macros to `%FARPROFILE%\Macros\internal` (`C:\Users\user\AppData\Roaming\Far Manager\Profile\Macros\internal`).
